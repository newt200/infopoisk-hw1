У меня получился csv-фал `woman.ru – 9 topic.csv` с ~ 57k строк.

## Структура проекта

### `preprocessing_data.py`

Файл отвечает за **предобработку текстового корпуса**.

Реализованные шаги:

- очистка текста
- токенизация
- удаление стоп-слов
- лемматизация с помощью `Mystem`
- формирование колонки `preprocessed_text`

Результатом работы является CSV-файл с предобработанным текстом, который используется для построения индексов.

---

### `inverted_index.py`

Файл содержит реализацию **обратного индекса** и поиска документов.

Реализованы:

- частотный обратный индекс
- BM25 индекс
- поиск документов по запросу
- ручная реализация индекса через словари

### `matrix_index.py`

Файл содержит **матричную реализацию индекса**.


### `main.py`

Файл содержит **единую точку входа в проект**.

Он объединяет:

1. предобработку корпуса
2. построение индекса
3. выполнение поиска

Основная функция проекта: `run_search()`

Она позволяет запустить поиск одним вызовом.

---

## Пример запуска и вывода результатов

```python
from main import run_search


results = run_search(
    query="Гадалка сказала, что мой муж мне изменяет",
    input_path="woman.ru – 9 topic.csv",
    preprocess=True
)
print("Результаты поиска:")
for _, row in results.iterrows():
    print("=" * 80)
    print(f"doc_id: {row['doc_id']}")
    print(f"score : {row['score']:.4f}")
    print()
    print(row["text"])
    print()
```
