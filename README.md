У меня получился csv-фал `woman.ru – 9 topic.csv` с ~ 57k строк.
Для работы беру только 30% данных, чтобы быстрее работало все.


## Предобработка данных

Предобработка корпуса реализована в файле **`preprocessing_data.py`**.  
На вход подается CSV-файл с текстами комментариев.

Для каждого текста выполняются следующие шаги:

1. приведение текста к нижнему регистру;
2. удаление пунктуации и лишних символов;
3. лемматизация с помощью библиотеки **Mystem** (`pymystem3`);
4. удаление стоп-слов;
5. фильтрация коротких токенов и чисел.

В результате для каждого документа сохраняются очищенный текст, список лемм и итоговая строка из лемм, которая используется для построения индексов и поиска.
